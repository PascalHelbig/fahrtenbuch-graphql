## Installation

### Entwicklungsumgebung
1. Node.js installieren (min 5.x)
2. Alle dependencies laden ```npm install```
3. ```.env``` Datei erstellen und mit Parametern befüllen (am einfachsten ist es ```.env-example``` als Vorlage zu benutzen)
4. (optional) mit ```npm start``` testen, ob die Anwendung startet. (Über die Route "\graphql" wird ein Test Client erreicht)

### Produktiv
1. Node.js installieren (min 5.x)
2. Alle dependencies laden ```npm install --production```
3. ```.env``` Datei erstellen und mit Parametern befüllen (am einfachsten ist es ```.env-example``` als Vorlage zu benutzen)
4. mit ```npm start``` den Server starten
